# Vanilla Features
## Vanilla is made to be as stock as possible adding in OMS for people who don't want all the bells and whistels of a major custom rom
- Backuptool
- Battery light
- Built in Flash kernel
- Built in GBoard
- Built in PixelLauncher
- Custom carrier settings 
- Custom navbar dimensions 
- Debloated AOSP
- f2fs support
- Google Assistant
- Notification light
- OMS
- Passes safetynet
- Rootless substratum
- Stock navbar editor
- Themeable battery
- Ticker notifications
- Toggleable NavBar
- Toggleable pixel animation
- More to come!
